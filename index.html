Jenkins任务模版-PHP版

大部分互联网产品更新升级频率很高，他们的环境（例如基本行为和尺寸）很少变化，痛苦的昨天一去不复返，如今互联网环境中不仅限于开发和部署产品，还偏重于产品内在品质的监控和可持续提升。

Jenkins 是领先的 开源 可持续集成 服务器。多亏它优秀的插件机制，它几乎可以支持构建、测试任何项目。

本项目的目标是提供php项目使用Jenkins创建任务的标准模版

只须遵照如下4个步骤进行就行：

  1. 安装Jenkins 插件 和 PHP 组件
  2. 用 Apache Ant构建PHP组件
  3. 用Jenkins任务模板配置 PHP组件
  4. 为你的php项目创建一个Jenkins任务
